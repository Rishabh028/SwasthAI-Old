# Swasth AI - Health Navigator

A comprehensive healthcare web application built with React.

## Installation

1. Make sure you have Node.js installed (v16 or higher)
2. Install dependencies:
```bash
npm install
```

## Running the App

Start the development server:
```bash
npm run dev
```

The app will automatically open at `http://localhost:3000`

## Building for Production

Create an optimized build:
```bash
npm run build
```

Preview the production build:
```bash
npm run preview
```

## Project Structure

```
SwasthAI/
├── Pages/              # All page components
├── Components/         # Reusable UI components
├── Entities/          # Data models
├── layout.js          # Main layout component
├── App.jsx            # App routing and setup
├── main.jsx           # Entry point
├── index.html         # HTML template
├── package.json       # Dependencies
└── vite.config.js     # Vite configuration
```

## Features

- **Home Dashboard** - Quick access to health information
- **Symptom Checker** - AI-powered symptom assessment
- **Appointments** - Book and manage doctor appointments
- **Health Records** - View medical history
- **Lab Tests** - Book and track lab tests
- **Pharmacy** - Order medications
- **Doctor Finder** - Search and book doctors
- **Health Tips** - Daily health advice

## Environment Setup

Make sure all required Node modules are installed before running the app.
