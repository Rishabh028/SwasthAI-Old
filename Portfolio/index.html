FinFlow - IoT‑enabled smart pond monitoring system

GitHub (Main) GitHub (Electronics) GitHub (Club)

Project Overview

Finflow is an IoT‑enabled smart pond monitoring system that automatically tracks multiple water quality parameters and alerts farmers when values exceed safe thresholds.

Multi-Parameter Sensing:

  • Ammonium ion concentration (via Berthelot reaction)
  • pH measurement
  • Dissolved Oxygen (O₂) level sensing

Sensor Suite

Ammonium ions
AS7265x Spectrometer
Colorimetric detection (Berthelot method)
pH
Analog pH Sensor (e.g., SEN0161)
Measures acidity/alkalinity
Dissolved Oxygen
Gravity Analog DO Sensor
Assesses oxygen levels for aquatic life

Enhanced Workflow

1

Water Intake

Pump draws pond water into analysis chamber

2

Sensor Suite Activation

  • pH sensor reads acidity level
  • DO sensor determines oxygen concentration
  • AS7265x detects ammonium ions via titration
3

Data Processing

  • Parameters time-stamped via NTP
  • Values uploaded to cloud via NodeMCU
  • Threshold breaches trigger farmer alerts

Technical Summary

Microcontrollers

Arduino Mega, NodeMCU

Connectivity

Wi-Fi, NTP, Cloud logging

Sensors

AS7265x, pH sensor, DO sensor

Alert System

Real-time notifications

Impact

Helps fish farmers prevent:

Water toxicity

Hypoxia conditions

pH imbalance
